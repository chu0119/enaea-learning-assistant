# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within ENAEA Learning Assistant, please send an email to the maintainer. All security vulnerabilities will be promptly addressed.

## Security Best Practices

When using this script:

1. **Keep your browser updated** to the latest version
2. **Only install from trusted sources** (GitHub or Greasy Fork)
3. **Review the code** before installing if you have concerns
4. **Report any suspicious behavior** immediately

## Data Privacy

This script:
- Does NOT collect or transmit any personal data
- Does NOT send data to external servers
- Only stores data locally in your browser (localStorage)
- All data remains on your local machine
