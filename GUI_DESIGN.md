# ENAEA Learning Assistant - GUI 面板设计方案

## 📋 设计概述

### 设计目标

1. **用户体验**：提供直观、易用的操作界面
2. **信息展示**：清晰显示当前状态和进度
3. **功能控制**：方便用户调整设置和控制脚本
4. **美观设计**：现代、简洁的 UI 设计

### 设计原则

- **简洁性**：避免信息过载，突出重点
- **一致性**：保持与平台风格一致
- **可访问性**：支持键盘操作和屏幕阅读器
- **响应式**：适配不同屏幕尺寸

## 🎯 功能模块详解

### 1. 状态栏

**位置**：顶部

**显示内容**：
- 脚本运行状态（🟢 运行中 / 🔴 已停止 / 🟡 暂停）
- 当前课程/模块名称
- 总体进度百分比

**交互**：
- 点击状态图标可快速切换运行/暂停

### 2. 视频控制区

**位置**：左侧上方

**功能按钮**：
| 按钮 | 功能 | 快捷键 |
|------|------|--------|
| ▶️ 播放 | 播放当前视频 | Space |
| ⏸️ 暂停 | 暂停当前视频 | Space |
| ⏭️ 下一个 | 跳转到下一个视频 | → |
| ⏮️ 上一个 | 跳转到上一个视频 | ← |

**速度控制**：
- 预设按钮：1x, 1.5x, 2x, 3x, 4x
- 自定义输入框：支持 0.5x - 16x

**音量控制**：
- 音量滑块：0% - 100%
- 静音按钮：快速静音/取消静音

### 3. 课程进度区

**位置**：右侧上方

**显示内容**：
- 进度条：可视化显示当前进度
- 当前课程：显示正在学习的课程
- 已完成统计：已完成/总课程数

**模块列表**：
- ☑️ 已完成模块（绿色勾选）
- 🔄 进行中模块（蓝色旋转图标）
- ⬜ 未开始模块（灰色）

**交互**：
- 点击模块可展开/收起课程列表
- 点击课程可快速跳转

### 4. 设置区

**位置**：中间

**开关选项**：
| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 自动播放 | ✅ 开启 | 视频完成后自动播放下一个 |
| 自动静音 | ✅ 开启 | 自动静音视频 |
| 弹窗处理 | ✅ 开启 | 自动处理系统弹窗 |
| 调试模式 | ❌ 关闭 | 显示详细日志 |

**下拉选项**：
- 检查间隔：1秒, 2秒, 3秒, 5秒, 10秒

### 5. 操作区

**位置**：中间下方

**按钮功能**：
| 按钮 | 功能 | 确认 |
|------|------|------|
| 🔄 重置进度 | 清除所有进度记录 | 需要确认 |
| 📤 导出配置 | 导出当前配置为 JSON | 无需确认 |
| 📥 导入配置 | 从 JSON 导入配置 | 需要确认 |
| 📊 统计 | 查看学习统计 | 无需确认 |

### 6. 日志区

**位置**：底部

**显示内容**：
- 时间戳
- 日志类型图标
- 日志消息

**日志类型**：
| 图标 | 类型 | 说明 |
|------|------|------|
| 📌 | info | 一般信息 |
| ✅ | success | 成功操作 |
| 🎯 | action | 执行操作 |
| ❌ | error | 错误信息 |
| ⚠️ | warning | 警告信息 |
| 🔍 | debug | 调试信息 |

**操作按钮**：
- 清除：清空日志
- 复制：复制日志到剪贴板

## 📱 响应式设计

### 桌面端 (>1024px)

- 完整布局
- 侧边栏可折叠
- 日志区可展开/收起

### 平板端 (768px - 1024px)

- 单列布局
- 控制区和进度区并排
- 日志区固定高度

### 移动端 (<768px)

- 单列布局
- 控制区和进度区堆叠
- 日志区默认收起
- 底部导航栏

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Space` | 播放/暂停 |
| `→` | 下一个视频 |

| `↑` | 增加音量 |
| `↓` | 减少音量 |

## 🔧 技术实现

### 技术栈

- **HTML5**: 页面结构
- **CSS3**: 样式和动画
- **JavaScript**: 交互逻辑
- **GM_addStyle**: 样式注入
- **GM_setValue/GM_getValue**: 数据持久化

### 文件结构

```
gui/
├── index.html          # 主页面（模板）
├── styles.css          # 样式文件
├── app.js              # 主逻辑
├── components/         # 组件目录
│   ├── statusBar.js    # 状态栏
│   ├── videoControl.js # 视频控制
│   ├── progress.js     # 进度显示
│   ├── settings.js     # 设置
│   ├── actions.js      # 操作
│   └── log.js          # 日志
└── utils/              # 工具函数
    ├── storage.js      # 存储
    └── helpers.js      # 辅助函数
```
